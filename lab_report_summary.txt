
ЧАСТЬ 1 — Токи смещения:
  Формула: Iвх = |Uвх| / R
  Iвхи = |−79.75 мкВ| / 1 кОм = 79.75 нА
  Iвхн = |−79.75 мкВ| / 1 кОм = 79.75 нА
  Вывод: значения соответствуют паспортным данным UA741 (80–500 нА)

ЧАСТЬ 2 — Передаточная характеристика:
  Принцип: ОУ без ОС работает как компаратор
  При Uвхи = 5 В: переключение при Uвхн ≈ 5.00 В
  При Uвхи = -3 В: переключение при Uвхн ≈ -3.00 В
  Графики: part2_transfer_Uvhi_5V.png, part2_transfer_Uvhi_neg3V.png

ЧАСТЬ 3 — Повторитель:
  Kидеал = 1 (0 дБ), частота среза ≈ 115 кГц
  График АЧХ: part3_bode_plot.png

ЧАСТЬ 4 — Неинвертирующий усилитель:
  Формула: K = 1 + RN/RVR3 → RVR3 = RN/(K-1), RN = 46.5 кОм
  • При K = 9:  RVR3 = 46.5 / 8 = 5.81 кОм
  • При K = 4:  RVR3 = 46.5 / 3 = 15.50 кОм
  График АЧХ: part4_bode_plot.png

ЧАСТЬ 5 — Инвертирующий усилитель:
  Формула: |K| = RN/RVR3 → RVR3 = RN/|K|, RN = 46.5 кОм
  • При |K| = 1.5:  RVR3 = 46.5 / 1.5 = 31.00 кОм
  • При |K| = 0.7:  RVR3 = 46.5 / 0.7 = 66.43 кОм
  График АЧХ: part5_bode_plot.png

ОБЩИЙ ВЫВОД:
  Экспериментальные характеристики ОУ UA741 согласуются с теоретическими моделями.
  Ограниченная полоса пропускания (~100–300 кГц при K>1) обусловлена внутренней
  частотной компенсацией микросхемы и соответствует паспортным данным.
